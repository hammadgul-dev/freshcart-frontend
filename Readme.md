# FreshCart — Full-Stack E-Commerce Web Application

A modern, responsive full-stack e-commerce platform built with the **MERN Stack**, featuring real-time cart management, JWT authentication, and a seamless shopping experience.

## Features

- **User Authentication** — Secure signup and login with JWT (JSON Web Tokens)
- **Email Notifications** — Automated confirmation emails via Nodemailer
- **Dual Product Catalog** — Browse both Grocery and Products from a single platform
- **Smart Search** — Real-time search across products and groceries
- **Advanced Filtering** — Filter by Brand, Category, and sort by price (High to Low / Low to High)
- **Cart Management** — Add, remove, and manage cart items with persistent state
- **Favorites** — Save and remove favorite products
- **Product Detail Page** — View full product information and specifications
- **Dark / Light Mode** — Theme toggle for comfortable browsing
- **Cloud Image Storage** — Product images hosted on Cloudinary CDN for fast delivery
- **Fully Responsive UI** — Optimized for desktop, tablet, and mobile devices

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js, Redux Toolkit, TanStack Query |
| Backend | Node.js, Express.js |
| Database | MongoDB Atlas |
| Authentication | JWT (JSON Web Tokens) |
| Image Storage | Cloudinary |
| Email Service | Nodemailer |
| External API | DummyJSON (Products) |
| Deployment | Vercel |

---

> **Note:** On first visit, grocery items may take a few seconds to load due to server cold start on free hosting. Refresh the page if items do not appear immediately.
